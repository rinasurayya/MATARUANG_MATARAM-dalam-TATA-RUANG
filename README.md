# MATARUANG merupakan Peta Tematik Digital berbasis WEB yang menyajikan informasi Peta Lahan Sawah Dilindungi (LSD), Peta Rencana Detil Tata Ruang (RDTR), Peta Rencana Tata Ruang Wilayah (RTRW) secara informatif dan mudah diakses, sebagai upaya Kantor Pertanahan Kota Mataram untuk meningkatkan kualitas tata kelola melalui keterbukaan dan kemudahan informasi pertanahan.
